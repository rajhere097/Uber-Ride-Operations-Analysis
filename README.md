🚖 Uber Ride Operations Analysis
📊 Cancellation, Revenue & Demand Insights
📌 Overview

This project analyzes 150K+ Uber ride bookings to identify cancellation drivers, demand patterns, and revenue loss using SQL, Python, statistical testing, machine learning, and Power BI dashboards.

The objective is to uncover operational inefficiencies and provide data-driven insights to reduce ride cancellations and improve revenue.

🛠 Tools & Technologies

🐍 Python – Pandas, Scikit-learn, Matplotlib
🗄 SQL (MySQL) – Data exploration & analysis
📊 Power BI – Interactive dashboards
📈 Statistics – Hypothesis testing
🤖 Machine Learning – Logistic Regression

🔑 Key Insights

✔ 25% of rides were cancelled
💰 Estimated ₹18.9M revenue loss due to cancellations
📍 Certain locations and time slots have higher cancellation rates
🚗 Ride distance and operational factors influence cancellations

🤖 Machine Learning Model

A Logistic Regression model was trained to predict ride cancellations.

📈 Model Performance

ROC-AUC Score: 0.95

The model shows strong ability to distinguish between cancelled and completed rides.
